# Network Scanning Automation

## Description
This project automates network scanning using Python:
- Ping Scanner
- ARP Scanner
- Nmap Scanner

## Requirements
- Python 3.x
- Nmap installed

## Installation

### Install Nmap
https://nmap.org/download.html

## How to Run

### Ping Scanner
python ping_scanner.py

### ARP Scanner
python arp_scanner.py

### Nmap Scanner
python nmap_scanner.py

### Unified Tool
python network_scanner.py

## Features
✔ Single & Multiple Host Ping  
✔ ARP Table Extraction  
✔ Multiple Nmap Scan Types  
✔ Save Results to File  
✔ Error Handling  
✔ Cross-platform Support  

## Example

Enter IP: 127.0.0.1  
Output: Host reachable
